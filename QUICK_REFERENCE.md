## Quick Commands

```bash
# Start server
cd /home/user/webapp
npm run pm2:start

# Check status
pm2 status

# View logs
pm2 logs resumeai --nostream

# Restart
pm2 restart resumeai

# Stop
pm2 stop resumeai

# Test API
curl http://localhost:3000/api/health
```
